# todo

archetypal todo process

## dev dependencies

you'll need:

- make
- go >= 1.12
- docker
- docker-compose

## running the server

1. clone this repository
2. run `make dev`
3. [http://localhost](http://localhost)

## working on the frontend

1. run `make dev`
2. in a different terminal window, run `npm run autobuild`
